Events API → Calendar

backend architecture + API integration · Python / Vercel

A live service syncing a rate-limited events API to a fast public calendar. The signal is an architecture decision under constraint: a daily cron writes one snapshot the read endpoint serves instantly.

Code only — live customer data is not included. Credentials load from environment variables.
